### Overview

* 總覽所有熱門飲品
* 依照喜愛的分類搜尋，例如:啤酒、Lounge Bar

### Drink Detail

* 瀏覽各個調酒的資訊，包含基酒、內容物、價格、酒吧及評論等資訊
* 點擊愛心即可加入收藏清單，點擊鉛筆即可新增評價

<img src="https://github.com/YuhsinLiao1995/Mr.9/blob/refactor/ScreenShots/detail_drink.png" width="200" height="420"></a>

### Bar Detail

* 瀏覽各酒吧的介紹，包含特色說明、地址、營業時間、電話、Menu及評價等
* 點擊愛心即可加入收藏清單，點擊鉛筆即可新增評價

<img src="https://github.com/YuhsinLiao1995/Mr.9/blob/refactor/ScreenShots/detail_bar.png" width="200" height="420"></a>

### Friends

* 使用者可查看好友動態，進而連結進入好友評價的飲品
* 使用者可以email搜尋好友或是欲追蹤之的對象

## Frameworks & Tools

* Design Pattern：`MVVM`、`Singleton`、`Observer`<br>
* Implementation：`RecyclerView`、`ChipGroup`、`Login (Google SDK、Firebase Authentication)`<br> 
* Jetpack：`ViewModel`、`LiveData`、`Lifecycle`、`Data Binding`、`Navigation`<br>
* Firebase：`FireStore`、`Firebase Crashlytics`、`Firebase Ayalytics`<br>
* Test：`JUnit`、`Mockito`、`Espresso`
